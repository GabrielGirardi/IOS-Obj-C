Principais diferenças entre C++ e Objetive-C

Objective-C C++
Objective-C é uma linguagem de programação orientada a objetos de propósito geral que adiciona sintaxe e semântica que permite uma linguagem orientada a objetos e se comporta como um superconjunto da linguagem de programação C. C ++ é uma linguagem de programação de uso geral, de nível médio, orientada a objetos e procedural. Também era conhecido como "C com Classes". É uma extensão da linguagem de programação C.
Em 1980, foi desenvolvido por Brad Cox e Tom Love na empresa Productivity Products International (PPI). Em 1980, foi desenvolvido por Bjarne Stroustrup nos laboratórios Bell.
Ele não oferece suporte a herança múltipla. Suporta herança múltipla.
No Objective C, Structs e classes não são tratados da mesma forma. Em C++, Structs e classes são tratados da mesma forma.
Em Objective C, para chamar as funcionalidades implementadas, são utilizadas mensagens. Em C ++, para chamar as funcionalidades implementadas, são utilizados métodos.
